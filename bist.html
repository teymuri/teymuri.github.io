Any (2023)

for any number of performers / für jede Anzahl von Musiker*innen

D.: var. (ca. 2' - 27')

Der Grundgedanke dieser Komposition ist eine endlose Variation. Die Abfolge von acht verschiedenen Klangwesen und die Reihenfolge der Buchstaben in einer Reihe von 26 Wörtern sind hier eng miteinander verknüpft. Jedes Klangwesen wird durch einen bestimmten Buchstaben repräsentiert, dessen Auftreten durch die Buchstabenreihenfolge in den Wörtern bestimmt wird. Die Aufführung erfolgt aleatorisch, wobei der Rhythmus des Buchstabierens jedes Wortes von den Aufführenden bestimmt wird. Der Aufbau der Musik geschieht zugleich aus einer Art Zustandsmaschine, die als Input eine Reihe von Buchstaben erhält und ein Wort ausgibt, das ausschließlich aus diesen Buchstaben besteht. Die Buchstaben des neu generierten Wortes dienen dann in jeder Iteration als neue Eingabe für die nächste Generation eines weiteren Wortes. Dabei können neue Buchstaben/Charaktere in einem vordefinierten Umfang hinzukommen oder ausfallen. Für die Wortgenerierung wurde ein trainiertes Sprachmodell verwendet.
The main idea of this composition is an endless variation. The sequence of eight different sound entities and the order of letters in a series of 26 words are closely linked. Each sound entity is represented by a specific letter, and its occurrence is determined by the order of letters in the words. The performance is aleatoric, with the rhythm of spelling each word determined by the performers. The music is built using a type of state machine that takes a sequence of letters as input and outputs a word consisting solely of those letters. The letters of the newly generated word serve as input for the next generation of a new word in each iteration, where new letters/characters can be added or omitted within predefined limits. A trained language model was used for the word generation.
